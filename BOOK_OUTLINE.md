# Breathing Room for Statistics: handbook outline

**Title:** Breathing Room for Statistics
**Subtitle:** A statistical handbook for respiratory research: from trials to omics and prediction
**Tagline:** With reproducible R examples
**Edition:** Single-volume handbook (Ch 0–22)

---

## Book structure (8 parts)

| Part | Chapters | Content |
|------|----------|---------|
| **I Foundations** | 0–2 | Preface, statistical thinking, data types |
| **II Description & comparison** | 3–4 | Descriptives, t-tests, ANOVA, nonparametric, proportions |
| **III Regression** | 5–7 | Linear, GLM, model building, LASSO |
| **IV Validation & prediction** | 8–9 | Reporting, bootstrap, ML metrics |
| **V Discovery (core)** | 10–12 | PCA, clustering, CASTOR case studies |
| **VI High-dimensional biology** | 13–16 | DE/FDR, batch, flow, antibody screens |
| **VII Integrated capstone** | 17 | CASTOR-HD pipeline + elastic net |
| **VIII Longitudinal & causal** | 18–22 | Mixed models, survival, missing data, IPW, mediation |
