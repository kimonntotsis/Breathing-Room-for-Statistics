# Figure index

All figures are generated by `R/examples/generate_figures.R` (and chapter scripts), except where noted. Regenerate after data changes:

```r
source("R/00_setup.R")

source("R/generate_data.R")
source("R/examples/generate_figures.R")
source("R/examples/generate_viz_pairs.R") # right vs wrong pairs only
```
