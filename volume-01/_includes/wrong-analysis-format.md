### Wrong-analysis box (handbook format)

Use this four-part structure in chapters and slides:

| | |
|---|---|
| **What went wrong?** | Name the analysis mistake in one line |
| **Why it matters** | Estimand, bias, or misleading clinical read |
| **Better approach** | Prespecified alternative method or reporting rule |
| **What to report** | Effect + uncertainty + denominators + limits |

Prioritise the **two highest-risk mistakes** in chapter main text; extended catalogues belong in appendices.
