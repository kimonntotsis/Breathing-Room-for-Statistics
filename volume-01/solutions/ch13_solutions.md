# Chapter 13 Solutions

**E13.1** Under the global null, about 5% of tests are expected to be nominally significant by chance (≈50 of 1000). Without multiplicity control, many "hits" are false positives.

**E13.2** Effect size (or log fold-change), uncertainty (95% CI), and multiplicity-adjusted evidence (q-value). Also report n used per feature when missingness matters.

**E13.3** When preprocessing, batch, or LOD handling is not described; when used as causal proof; when no confirmation or replication plan exists.

**E13.4** Counts are non-negative integers with mean–variance relationship; Gaussian tests ignore library size and overdispersion.

**E13.5** Multiplicity: many nominal significances are expected under the null; BH adjustment leaves no FDR-controlled discoveries. Report effect sizes anyway.

**Applied (outline)** Proteomics often 0 at q<0.05 with batch; RNA many in teaching global-shift demo. Draft honest Results for both.
